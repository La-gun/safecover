How MicroInsurancePolicy.sol is deployed when a policy is bound

Contract: MicroInsurancePolicy.sol

constructor(address _insured, uint256 _premium, string memory _coverage)
  - insurer = msg.sender
  - insured = _insured
  - premium = _premium (wei)
  - coverage = _coverage
  - active = true
  - emit PolicyBound(insured, premium, coverage)

Execution flow

When you bind a policy, the backend simulates deploying this contract. In production, ethers.js would call the factory or deploy directly.
